# 🧊 Запуск
Для запуска **Austrum** вам необходимо
установить [Bun - Javascript Runtime](https://bun.sh)

Дальше пропишите следующую
команду в терминал:

```bash
bun run dev
```

Если вы хотите запустить **Austrum** не в режиме
разработчика, то пропишите эту команду:

```bash
bun run start
```
